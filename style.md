# Front-end Style Guide

## Layout

The designs were created to the following widths:

- Mobile: 375px
- Desktop: 1440px

## Colors

### Neutral
- black (main neutral): #000
- white: #fff

### SVG 
- purple: #6c83ff
- beige-orange: #ffa98e;

### Link:
- when hover, square bracket appears around the text(indicate it is getting hover)
